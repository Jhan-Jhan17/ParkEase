# ParkEase — Parking Management System

**Short description**
A web-based parking management system (built with Vite + TypeScript). This repository contains the full source code for the ParkEase project.

## Contents
- `index.html` — main entry file for the web app. :contentReference[oaicite:1]{index=1}
- `src/` — application source code (components, services, styles).
- `package.json` / `vite.config.ts` — build & dependency configuration. :contentReference[oaicite:2]{index=2}

## Requirements
- Node.js 18+ (or LTS)
- npm (or yarn)

## Setup & Run (development)
```bash
# clone (if not already)
git clone https://github.com/Jhan-Jhan17/ParkEase.git
cd ParkEase

# install
npm install

# run dev server
npm run dev
